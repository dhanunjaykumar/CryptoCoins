# CryptoCoins

#Features
1. Fetch & show the crypto coins from API using Async-awit
2. Search Coins
3. Filter Coins
4. MVVM architecture
5. Custom search bar with hide & show feature
6. Custom tags list to show Filters
7. Custom tag to show whether tag is seleceted or not
